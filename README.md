# Work Requirement 3

## Python task

This project uses Python's `socket` module to resolve a hostname into an IP address and print the result.

## Reflection

In this task, I learned more about how TCP/IP, sockets, browsers, HTTP, and web servers work together. I understood better that a browser acts as a client and sends HTTP requests to a server, which then sends back a response. I also saw how the socket API gives access to lower-level networking functions in Python. What surprised me was how much of the internet depends on simple concepts like IP addresses, ports, and requests. I would like to explore HTTP requests and responses further, especially how browsers and servers communicate in practice.
